HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the structural backbone of the website.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class attribute allows us to use a CSS rule to any element when added to the HTML.

Git

  • Git and GitHub are used for version control.
  • To merge branches to main branch you need to use a Pull Request.
  • Issues are considered as a to-do list.

JavaScript

  • Code is executed from top to bottom in JavaScript.
  • Controlflow can be interupted using "if" statements.
  • JavaScript allows users to interact with webpages.
